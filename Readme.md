# 📌 Project Overview

This project analyzes the **global data jobs market**, focusing on salary trends, in-demand skills, job distribution, and experience-level insights. The dashboard is built using **Power BI** to provide interactive visualizations that help uncover patterns in the data industry.

# 📊 Key Insights & Findings

## 💰 Salary Insights

* The **average salary** across all roles is approximately **$120,000 per year**.
* **Data Engineers** emerge as the highest-paid role, with average salaries exceeding **$140,000**.
* Entry-level positions earn around **$60,000–$80,000**, while senior roles exceed **$160,000+**.
* There is a **salary gap of over 2x** between entry-level and senior professionals.

📌 *Insight:* Specializing and gaining experience significantly increases earning potential.

---

## 🧠 Skills Analysis

* The most in-demand skills include:

  * **SQL**
  * **Python**
  * **Power BI / Tableau**
  * **Excel**
* High-paying niche skills:

  * **PySpark** (~$200K average salary)
  * **Machine Learning** (~$180K)
  * **Cloud technologies (AWS, Azure)** (~$170K)

📌 *Insight:* Advanced and big data skills command significantly higher salaries.

---

## 📈 Job Demand Trends

* **Data Analyst roles** have the highest number of job postings (~40% of total jobs).
* **Data Scientist roles** account for ~30%.
* **Data Engineers** make up ~20% but offer higher salaries.

📌 *Insight:* Data Analyst roles are easier to enter, but engineering roles are more lucrative.

---

## 🌍 Geographic Insights

* The **United States** dominates the job market with the highest salaries.
* Remote jobs account for a significant portion (~25–35%).
* European and Asian markets show growing demand but slightly lower salary ranges.

📌 *Insight:* Location plays a major role in salary differences.

---

## 🧑‍💼 Experience-Level Analysis

* Entry-level (0–2 years): Lower salaries but high demand
* Mid-level (3–5 years): Significant salary growth (~40–60% increase)
* Senior-level (5+ years): Highest compensation and leadership roles

📌 *Insight:* Career progression in data roles is strongly tied to experience.
